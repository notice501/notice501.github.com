每日vim插件--Gist.vim

今天介绍的插件Gist.vim能够在vim中方便的创建和查看gist。

gist我就不过多介绍了，github提供的一个代码片段托管服务。不太了解的同学可以看看这个教程

要使用这个插件需要安装ygie依赖插件：

确保在git的global配置中设置的是github用户名：

$ git config --global github.user <username>

该插件在首次使用时会需要你输入github密码来获取token，并将其保存在~/.gist-vim.
